単体法 7日間練習問題

1日1問ずつ、スラック変数・単体表・ピボット操作を練習します。

学習の進め方：
① スラック変数を入れる
② 初期単体表を作る
③ z行で一番マイナスの大きい列を選ぶ
④ 比テストで出る変数を決める
⑤ ピボット操作をする
⑥ z行にマイナスがなくなれば終了
Day 1：基本の最大化問題
最大化　z = 3x₁ + 2x₂
x₁ + x₂ ≤ 4
2x₁ + x₂ ≤ 6
x₁, x₂ ≥ 0
ヒント：まず s₁, s₂ を入れて等式に直しましょう。
解答を見る
最適解は x₁ = 2, x₂ = 2
最大値は z = 10
Day 2：係数が少し大きい問題
最大化　z = 5x₁ + 4x₂
6x₁ + 4x₂ ≤ 24
x₁ + 2x₂ ≤ 6
x₁, x₂ ≥ 0
ヒント：z行は -5, -4 から始めます。最初は x₁ が入る候補です。
解答を見る
最適解は x₁ = 3, x₂ = 1.5
最大値は z = 21
Day 3：制約条件が3本ある問題
最大化　z = 4x₁ + 3x₂
x₁ + x₂ ≤ 5
2x₁ + x₂ ≤ 8
x₁ + 3x₂ ≤ 9
x₁, x₂ ≥ 0
ヒント：スラック変数は s₁, s₂, s₃ の3つです。
解答を見る
最適解は x₁ = 3, x₂ = 2
最大値は z = 18
Day 4：x₂から入る練習
最大化　z = 2x₁ + 5x₂
x₁ + x₂ ≤ 6
x₁ + 2x₂ ≤ 8
3x₁ + 2x₂ ≤ 18
x₁, x₂ ≥ 0
ヒント：z行では -5 の方が大きくマイナスなので、最初は x₂ が入ります。
解答を見る
最適解は x₁ = 4, x₂ = 2
最大値は z = 18
Day 5：端の点が最適になる問題
最大化　z = 6x₁ + 2x₂
x₁ + x₂ ≤ 7
2x₁ + x₂ ≤ 10
x₁ + 2x₂ ≤ 10
x₁, x₂ ≥ 0
ヒント：x₁の利益が大きいので、x₁方向に進む意識で表を見ましょう。
解答を見る
最適解は x₁ = 5, x₂ = 0
最大値は z = 30
Day 6：分数が出る問題
最大化　z = 7x₁ + 5x₂
3x₁ + 2x₂ ≤ 18
x₁ + 2x₂ ≤ 10
x₁, x₂ ≥ 0
ヒント：交点を求めると分数ではなく整数になりますが、単体表では途中で分数が出ます。
解答を見る
最適解は x₁ = 4, x₂ = 3
最大値は z = 43
Day 7：総合問題
最大化　z = 8x₁ + 6x₂
2x₁ + x₂ ≤ 14
x₁ + x₂ ≤ 10
x₁ + 3x₂ ≤ 18
x₁, x₂ ≥ 0
ヒント：制約が3本あるので、比テストでどの行が先にぶつかるかを丁寧に見ましょう。
解答を見る
最適解は x₁ = 6, x₂ = 2
最大値は z = 60
